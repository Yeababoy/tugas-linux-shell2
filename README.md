# Project Sistem Operasi

## Deskripsi
Project ini dibuat sebagai tugas praktikum Sistem Operasi.

Script `src/main.sh` digunakan untuk menghitung ukuran folder project ini secara otomatis dan menyimpannya ke file `data/sample.txt`.

## Struktur Folder
project_Sistem_Operasi_B/

├── src/ # berisi script shell (main.sh)

├── data/ # hasil output dari script (sample.txt)

├── doc/ # dokumentasi (jika diperlukan)

└── README.md # file penjelasan project


## Cara Menjalankan Script

```bash
./src/main.sh
